Favorite Projects

  • Critter World, CS 2112:
    In a group project, we created a simulator of a world full of critters. Critters had their own programmable language called
    Critter Lang that described their behavior and actions. Critters could even mate which would make a new critter with functioning code that was
    created from a combination of the two parents’ code. The project displayed all of this in a GUI using JavaFX which let the user step through the
    world’s time.
  • Uno, CS 3110:
    In a group project, we implemented a terminal-based Uno game in OCaml. We made a solo player Uno-Solitaire game and a
    multiplayer original Uno game. Uno could be played against other players or against AIs on 3 different difficulties. The state of any game could
    be saved and loaded to and from a file through commands.
  • Chess, Personal Project:
    As a personal project, I created a representation of Chess in Java that allows chess pieces to make valid
    moves, checks if the king is in check or checkmate, determines if a player is in stalemate, changes the piece scoreboard as the game goes on, checks
    if a piece is being protected and/or protecting another piece, and has a custom AI that uses MiniMax techniques. The project is displayed with JavaFX
